# Resource Intelligence

Production-ready **creative asset orchestration** for agents — icons, illustrations, photos, fonts, logos, avatars, 3D, patterns, animations, and mockups.

## Scope

- Discover, rank, and recommend assets across licensed provider ecosystems
- **Default catalog: commercial-use only** — non-commercial and mixed-NC providers excluded
- **License Intelligence** on every result — commercial use, attribution, redistribution, AI restrictions
- **Resource Graph** — knowledge about providers and assets, not copyrighted file hosting
- Planning + provider routing + ranking aligned with design system context

**Not in scope:** Permanent asset storage, CDN hosting, or redistributing third-party packs through MCP.

## Related modules

| Module | Relationship |
|--------|----------------|
| **Component Intelligence** | UI components vs raw assets |
| **Inspiration Intelligence** | Full-page design inspiration vs discrete assets |
| **Consistency Intelligence** | Design-system fit scoring for resource recommendations |
| **Figma Intelligence** | User's own Figma files vs public asset providers |
